Photo indexing details:
For each valid image:
- open with Pillow
- get width/height
- determine orientation
- compute average color efficiently, using resized image thumbnail
- compute brightness
- compute saturation
- store normalized values
- avoid loading enormous images at full size when possible
- catch errors and continue

Average color:
- Resize to a small thumbnail, such as 32x32.
- Convert to RGB.
- Compute average R/G/B.

Brightness:
- Use average luminance or simple formula:
  0.2126 R + 0.7152 G + 0.0722 B

Saturation:
- Simple HSV-based average or approximate max-min difference normalized.

Color matching:
Use simple color distance between photo average color and theme primary/accent colors.
It does not need to be perfect. It just needs to avoid obviously clashing images.

Hybrid wallpaper rendering:
For hybrid mode:
1. Load selected photo.
2. Crop/resize to exact target size.
3. Apply dark overlay based on theme background/surface.
4. Apply subtle color tint using theme primary/accent.
5. Add geometric linework consistent with display role.
6. Save as PNG.
7. Preserve no EXIF in output.

For local_photo mode:
- Still crop/resize and apply mild theme overlay.
- Keep photo more visible than hybrid mode.
- Save as PNG.
- Preserve no EXIF in output.
