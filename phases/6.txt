Display-specific behavior:

Display 4: Left Atmosphere
Purpose:
- mood-setting side monitor
- more atmospheric than the center
- can be slightly more expressive
- should still remain calm and work-safe

Style:
- ambient gradient
- subtle geometric overlays
- soft cyberpunk accents
- low to medium visual density based on ThemePlan
- no large text

Display 2: Center Command
Purpose:
- main work monitor
- lowest distraction
- ticket/work readability
- clean command-center background

Style:
- very sparse or sparse
- darker center area
- subtle grid or linework around edges
- no bright center glow
- no large shapes behind the center
- no text
- no clutter

Display 3: Right Inspiration
Purpose:
- more creative/inspirational private monitor
- can be slightly more dynamic than Display 2
- can use stronger theme accent shapes

Style:
- abstract geometry
- cyberpunk linework
- subtle motion-feel composition
- medium density only when ThemePlan allows
- no text

Theme constraints:
1. If ThemePlan visual_density is very_sparse:
   - use very few lines/shapes
   - keep opacity low
   - center monitor almost plain

2. If ThemePlan visual_density is sparse:
   - subtle grid and limited accent shapes

3. If ThemePlan visual_density is medium:
   - more geometry allowed on Displays 4 and 3
   - Display 2 stays sparse

4. If ThemePlan cyberpunk_intensity is low:
   - muted accents
   - soft glow
   - minimal grid

5. If ThemePlan cyberpunk_intensity is high:
   - stronger accent geometry on Displays 4 and 3
   - Display 2 remains controlled

6. If ThemePlan notes/body constraints include headache, sensory load, low sleep, fatigue:
   - reduce brightness
   - reduce density
   - avoid aggressive red/orange
   - avoid busy grids

7. Display 2 must always be the least visually noisy wallpaper.
